# Tuteur Éducatif Personnalisé basé sur un LLM

## Objectif
Ce projet vise à concevoir et déployer un **tuteur éducatif personnalisé**
basé sur un **Large Language Model (LLM)** pour accompagner les étudiants
de **Licence 3 Informatique**.

Le tuteur adapte ses explications en fonction :
- de la matière choisie
- du niveau de l’étudiant

---

## Matières prises en charge
- Programmation Python
- Algorithmique et structures de données

---

## ⚙️ Technologies utilisées
- Python
- Streamlit (interface web)
- Hugging Face Inference API (LLM open-source)
- GitHub (versionnement)
- Streamlit Cloud (déploiement)

---

## Fonctionnement
1. L’utilisateur choisit une matière et un niveau
2. Il pose une question
3. Le système génère un prompt pédagogique personnalisé
4. Le LLM fournit une explication adaptée
5. Une question finale vérifie la compréhension

---

## Déploiement
L’application est déployée en ligne via **Streamlit Cloud**.
